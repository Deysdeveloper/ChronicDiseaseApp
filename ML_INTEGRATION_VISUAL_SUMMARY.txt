┌───────────────────────────────────────────────────────────────────────────┐
│                     🔄 DETECTION ALGORITHM                                 │
├───────────────────────────────────────────────────────────────────────────┤
│                                                                           │
│   Step 1: Sliding Window                                                 │
│   ┌────────────────────────────────────────────────┐                    │
│   │ [v1, v2, v3, v4, v5, v6, v7, v8, v9, v10]     │                    │
│   │     [v2, v3, v4, v5, v6, v7, v8, v9, v10, v11]│                    │
│   │         [v3, v4, v5, v6, v7, v8, v9, v10, v11, v12]                │
│   └────────────────────────────────────────────────┘                    │
│                                                                           │
│   Step 2: Normalize (min-max)                                            │
│   ┌────────────────────────────────────────────────┐                    │
│   │ normalized = (value - min) / (max - min)      │                    │
│   └────────────────────────────────────────────────┘                    │
│                                                                           │
│   Step 3: TFLite Inference                                               │
│   ┌────────────────────────────────────────────────┐                    │
│   │ Input [1, 10, 1] → Model → Output [1, 10, 1] │                    │
│   └────────────────────────────────────────────────┘                    │
│                                                                           │
│   Step 4: Calculate MSE                                                  │
│   ┌────────────────────────────────────────────────┐                    │
│   │ MSE = mean((reconstruction - input)²)         │                    │
│   └────────────────────────────────────────────────┘                    │
│                                                                           │
│   Step 5: Voting Mechanism                                               │
│   ┌────────────────────────────────────────────────┐                    │
│   │ If MSE > threshold → votes[point]++           │                    │
│   └────────────────────────────────────────────────┘                    │
│                                                                           │
│   Step 6: Deviation Check                                                │
│   ┌────────────────────────────────────────────────┐                    │
│   │ localMedian = median(values[i-5:i+5])         │                    │
│   │ deviation = |value - localMedian|             │                    │
│   └────────────────────────────────────────────────┘                    │
│                                                                           │
│   Step 7: Final Decision                                                 │
│   ┌────────────────────────────────────────────────┐                    │
│   │ isAnomaly = (votes >= k_votes) AND           │                    │
│   │             (deviation >= delta OR            │                    │
│   │              value <= criticalLow OR          │                    │
│   │              value >= criticalHigh)           │                    │
│   └────────────────────────────────────────────────┘                    │
│                                                                           │
